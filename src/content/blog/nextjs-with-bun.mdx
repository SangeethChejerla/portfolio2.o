## Bun with Next.js: Essential Commands

Are you ready to supercharge your Next.js development with Bun? Here's a comprehensive guide to get you started!

### Creating a New Next.js Project

To create a new Next.js project using Bun, run:

```bash
bun x create-next-app@latest [folder-name]
```

### Installing Shadcn UI

Shadcn UI is a popular component library for building beautiful user interfaces. To install it in your project:

```
bun x --bun shadcn-ui@latest init
```

## Adding Shadcn UI Components

To add specific Shadcn UI components to your project:

```
bun x --bun shadcn-ui@latest add [component-name]
```

### Installing External Libraries

To add any external library to your project:

```
bun add [library-name]
```

### Running Your Next.js Application

To start your development server:

```
bun run dev
```
